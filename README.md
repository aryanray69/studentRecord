# Student Records
- A simple Student Records web application built using HTML, CSS and JavaScript.

## Functions used 
1. map()
2. filter()
3. addEventListener()
4. displayStudents()
5. toLowerCase()
6. includes()
7. join()
8. getElementById()

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run the Project 
1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Make sure all files are in the same folder.
4. Open index.html in your browser or you can use the live server extension.
